# Templates

## Objetivo

Plantillas estándar reutilizables para el proyecto AtlasERP.

## Contenido

- `task-detail-template.md` — plantilla estándar para archivos de task individuales.
- `domain-blueprint-template.md` — plantilla para blueprints de dominio.
- `technical-blueprint-template.md` — plantilla para blueprints técnicos.
- `adr-template.md` — plantilla para Architecture Decision Records.
- `ai-task-review-checklist.md` — checklist de revisión de tasks generadas por IA.
- `module-acceptance-checklist.md` — checklist de aceptación por módulo.
- `release-checklist.md` — checklist de release candidata.

## Convenciones

- Nombre de archivo en `kebab-case`.
- Cada plantilla debe ser autocontenida con instrucciones de uso.
- Mantener idioma español de México y codificación UTF-8.
