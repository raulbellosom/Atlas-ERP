# Visión de Producto en 3 Horizontes

## Horizonte 1 - Corto plazo (v1)
- Consolidar la plataforma base y gobierno técnico.
- Entregar Sync Core operativo con revisión humana de conflictos.
- Entregar Financial Operations Core funcional y auditable.
- Dejar capacidad de operación web y desktop con offline parcial.

## Horizonte 2 - Mediano plazo (v1.x - expansión controlada)
- Fortalecer reportes, observabilidad, CI/CD y continuidad operativa.
- Conectar Financial Operations Core con capacidades contables iniciales.
- Expandir controles de permisos finos y trazabilidad transversal.

## Horizonte 3 - Largo plazo (v2+)
- Incorporar módulos completos adicionales: Accounting Core, HR Core, Purchases, Inventory y CRM.
- Operar como plataforma empresarial modular integrada.
- Mantener crecimiento por contratos de módulo sin romper núcleo ni sincronización segura.
