# Project Vision

## Qué es
AtlasERP es una plataforma modular para operaciones empresariales con backend central, cliente web y cliente desktop, diseñada para crecer por módulos.

## Qué no es
No es un ERP completo en el primer release. El producto empieza por una base fuerte de plataforma y un primer macro-módulo de operación financiera.

## Superficies de uso
- Web App
- Desktop App
- Backend central

## Principio de crecimiento
Cada módulo debe poder convivir con futuros módulos sin reescrituras grandes.
