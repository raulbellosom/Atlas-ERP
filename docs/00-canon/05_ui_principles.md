# UI Principles

## Carácter no negociable
Estos principios son obligatorios para toda interfaz de AtlasERP.

- React + Vite + JavaScript
- TailwindCSS 4.1
- Sin Bootstrap
- Iconos: Lucide o Phosphor
- Diseño limpio, moderno y profesional
- Toda pantalla debe contemplar:
  - loading
  - empty
  - error
  - offline
  - sync pending
- El Sync Center debe ser visible y entendible

## Criterio de cumplimiento
Una pantalla no se considera terminada si no cumple todos los estados mínimos de UX.
