# Modular Strategy

## Núcleo inicial
- Auth
- Organizations
- Users
- Roles & Permissions
- Audit
- Attachments
- Settings
- Sync Core

## Primer macro-módulo
- Financial Operations Core (visible como Tesorería y Movimientos)

## Módulos futuros
- Accounting Core
- HR Core
- Purchases
- Inventory
- CRM
- Payroll

## Regla de ownership
Cada entidad tiene un módulo dueño. Otros módulos pueden leerla o vincularla, pero no re-definirla.
