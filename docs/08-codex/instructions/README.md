# Instructions

## Objetivo

Esta carpeta contiene instrucciones operativas específicas para la ejecución de tareas dentro de AtlasERP. Las instrucciones difieren de los prompts en que son directivas operativas concretas, no instrucciones generales de comportamiento.

## Contenido

- `CODEX_MASTER_INSTRUCTIONS.md` — instrucciones maestras globales (ubicado en `docs/08-codex/`).
- Instrucciones adicionales se agregarán conforme avance el proyecto.

## Convenciones

- Nombre de archivo en `kebab-case` con sufijo `-instructions.md`.
- Cada instrucción debe ser autocontenida y referenciar documentos de soporte.
- Mantener idioma español de México y codificación UTF-8.
